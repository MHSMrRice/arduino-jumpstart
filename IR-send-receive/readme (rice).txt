In order to easily send and receive IR codes, you will need to properly install the library that exists on github for doing so. Be  sure to Read The Full Manual there and follow instructions given by the library and its examples...

Github URL: https://github.com/cyborg5/IRLib2

Adafruit tutorial URL: https://learn.adafruit.com/using-an-infrared-library/overview

IR reciever hookup: https://learn.adafruit.com/ir-sensor/using-an-ir-sensor

The two sketches contained here are stripped out examples from the library, that are almost ready to pair for send & receive.
